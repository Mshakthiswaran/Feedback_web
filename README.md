# Teacher Feedback Collection System

A full-stack application built for academic institutions to collect structured feedback from students regarding their teachers and subjects.

## Features
- **Professor Dashboard**: Matrix-style form builder.
- **Student View**: Mobile friendly table layout.
- **Reporting**: Dynamic charts, export to PDF and Excel.

## Tech Stack
**Frontend:** React, Vite, TailwindCSS, Zustand
**Backend:** Node.js, Express, PostgreSQL, Prisma
